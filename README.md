# Benchmarking-VLMs-for-VIA-in-Pakistan [Work in Progress]
Benchmarking Vision-Language Models (VLM) for Visual Impairment Assistance in Pakistan using real-world local environments. This project evaluates VLMs like LLaVA-13B and introduces a human-centered, VI-oriented description format is focused on navigation, spatial awareness, safety, and accessibility.
